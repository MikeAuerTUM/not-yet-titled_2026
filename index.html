What is this website?

This is a website for the interactive installation "Not yet titled (Stars)" (2026) by Mike Auer at HfP.

Its goal is to add some background and context to the installation and possibly link to further resources and related material.

AI Use Transparency

Note for the Examiners of this Seminar

For the setup of this website, AI tools were used during the development process. AI was only used for the technical aspects of this website. All content was created without AI.

While I critically reflected on the use of AI in this way and found it to be appropriate here, this can of course be discussed.
